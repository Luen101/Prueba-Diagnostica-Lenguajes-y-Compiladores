# Prueba-Diagnostica-Lenguajes-y-Compiladores
Serie de problemas a modo de primera asignación de la UC Lenguajes y Compiladores en la UNEG

**Alumno:** Luis Millan

## Serie de Planteamientos.

### Planteamiento N1. Generación de Polinomios mediante el Triángulo de Pascal y Memoria Dinámica.
El primer planteamiento de la prueba diagnóstica consiste en desarrollar un programa capaz de gestionar polinomios de la forma (x+1)^n utilizando estructuras de datos dinámicas.

### Lenguajes Utilizados: [![C++](https://img.shields.io/badge/C++-%2300599C.svg?logo=c%2B%2B&logoColor=white)](#) y [![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=fff)](#)

### Planteamiento N2. Validación de Estructura de Datos en Notación FEN.
Se trata de la implementación de algoritmos que utilicen memoria dinámica para segmentar y procesar los seis campos críticos de la norma, que incluyen desde la disposición geométrica de las piezas en un tablero de ocho por ocho, hasta metadatos precisos como el turno del jugador, los derechos de enroque y la legalidad de los movimientos de peón al paso.

### Lenguaje Utilizado: [![C++](https://img.shields.io/badge/C++-%2300599C.svg?logo=c%2B%2B&logoColor=white)](#) 

### Planteamiento N3. Traducción y Análisis Léxico de Palabras Reservadas del Lenguaje C.
El tercer planteamiento de la prueba diagnóstica se enfoca en la manipulación de archivos y el análisis de cadenas para simular una de las fases iniciales de un compilador: el reconocimiento de tokens.

### Lenguaje Utilizado: [![C](https://img.shields.io/badge/C-00599C?logo=c&logoColor=white)](#) 

### Requisitos para ejecutar archivos de C y C++ 
   - Abrir `Problema1A.cpp` , `Problema_2.cpp` y `Problema3.c` en el entorno [![Dev c++](https://img.shields.io/badge/Dev_c++-red)](https://sourceforge.net/projects/embarcadero-devcpp/)
   - Compilar y correr usando la tecla `F11`

### Requisitos para ejecutar archivos de Python:
- Instalar ![Visual Studio Code](https://custom-icon-badges.demolab.com/badge/Visual%20Studio%20Code-0078d7.svg?logo=visualstudiocode&logoColor=white)
- Instalar la extensión oficial de [![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=fff)](#)
 (ms-python.python) para habilitar el botón de "Run" y la depuración.
 en su versión 3.x (preferiblemente 3.8 o superior). Puedes descargarlo desde python.org.  
- Puedes comprobar si está instalado abriendo una terminal o CMD y escribiendo python --version o python3 --version.
- Abrir archivo `Problema1B.py`
- Presionar `Ctrl` + `F5`

También puedes usar editores como PyCharm o simplemente un bloc de notas y ejecutarlo desde la terminal.
